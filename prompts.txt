This is the skeleton of an R package which allows the user to add CSV data download as 
a button to a given plotly chart in R.  In most cases, the charts will be generated
from ggplot output using the ggplotly() function in R.

* Please add comprehensive unit testing to this project to ensure that
the required button is indeed added to plotly charts to which this 
decoartor is applied and
* that the CSV output data matches the input data.

* Please ensure that this package is upto CRAN standards.

* Please check that the export button is actually present in 
the plotly output.

* Please write a comprehensive vignette for this package.

* Please add a test using a quarto document to ensure that the 
package works as expected in a quarto document.  Please ensure
to test the global decorator as well as the per-plot decorator.

* When using the global decorator, the default file name should
be taken from the figure or chunk name associated with the 
plotly imgae, by default.  If no such name is found, then
the default file name should be "plotly-data.csv".
